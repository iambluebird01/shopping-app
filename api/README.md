 Create .env file with following properties and paste config values respectively
   ```bash
   MONGO_URL = ------------------
   PASS_SEC = ------------------
   JWT_SEC = ------------------
   STRIPE_KEY = ------------------
   ```
 Install node packages
   ```bash
   - npm install
   or
   - yarn
   ```
 Start the development server
   ```bash
   npm start / npm run dev
   or
   yarn start / yarn run dev
   ```
